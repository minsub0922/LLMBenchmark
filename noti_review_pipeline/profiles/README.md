# profiles

This module stores reusable review-policy profiles.

## Responsibility
- human feedback driven rewrite preferences
- title quality profiles
- grammar cleanup profiles
- configurable policy bundles for dataset improvement

## Main file
- `feedback_profiles.py`

## Example
Typical profile names:
- `none`
- `title_grammar_v1`
- `title_content_coverage_v1`
- `title_grammar_coverage_v1`
